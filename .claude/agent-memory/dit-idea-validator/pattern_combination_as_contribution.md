# Pattern: Combination-as-contribution

When literature overlap shows every sub-mechanism is published and the only differentiators are (a) "systematic comparison of X", (b) "closed-form analysis of an existing score", (c) "ablation of axis Y" — this is **ablation-grade**, not contribution-grade.

**Why:** Top-tier reviewers explicitly downgrade papers that present a new bundle without a new insight. "Application of [X] to [DiT]" is the canonical reject reason. Three ablations stapled together does not clear the bar.

**How to apply:**
- When evaluating any DiT efficiency idea, first list the sub-mechanisms (score, scheduling, reactivation, CFG handling, recovery). If each is independently published, the burden of proof shifts to "what new thing is empirically true *because* of the combination" — not "what new thing exists."
- Push the user toward one of: failure-mode discovery, theoretical bound, novel signal source, or compositionality study (pruning x caching).
- Workshop venue is honest path forward when contribution is real but small.
